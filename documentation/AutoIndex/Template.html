		Constructor __construct (line 98)
	
 
	
	

Parses the text in $filename and sets the result to $out. We cannot  use ExceptionDisplay here if there is an error, since it uses the  template system.


Steps to parse the template: 
  • remove comments
  • replace {info} variables
  • replace {words} strings
  • replace {config} variables
  • include other files when we see the {include} statement
