		get_all_langs (line 66)
	
 
	
	

Returns a list of all files in $path that match the filename format  of language files.


There are two valid formats for the filename of a language file. The  standard way is the language code then the .txt extension. You can  also use the language code followed by an underscore then the  country code. The second format would be used for dialects of  languages. For example pt.txt would be Portuguese, and pt_BR.txt  would be Brazilian Portuguese. The filenames are case insensitive.
